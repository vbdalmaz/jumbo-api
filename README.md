# Backend Test

Testing I'm using Spring Boot with Java 8 and MongoDB as Database

# Docker 
To run local just install java 8 and docker

docker run --name some-mongo -d -p 27017:27017 mongo

Go to project folder and execute:
mvn spring-boot:run
